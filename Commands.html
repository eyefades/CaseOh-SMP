Minecraft Server Commands

General Commands

skipnight

Description: Cast a vote to skip the night

Alias: sn

Usage: /skipnight <yes/no>

vote

Description: Opens the voting GUI

Usage: /vote

rules

Description: Displays the server rules

Usage: /rules

help

Description: Opens the help GUI

Usage: /help

recipe

Description: Show the crafting recipe for an item

Usage: /recipe <item_name>

mail

Description: Manage your in-game mail

Usage:

  • /mail clear - Clear all mail
  • /mail read - Read your mail
  • /mail send - Send mail to a player

Teleportation Commands

spawn

Description: Teleport to the spawn point of the server

Usage: /spawn

rtp

Description: Open the random teleport GUI to choose your destination world

Usage: /rtp

warp

Description: Teleport to a specified warp location

Usage: /warp <warp_name>

crates

Description: Teleport to the crates area

Usage: /crates

marketplace

Description: Open the marketplace GUI

Alias: mp

Usage: /marketplace

home

Description: Teleport to a specific home

Usage: /home <home_name>

sethome

Description: Set a home at your current location

Usage: /sethome <home_name>

removehome

Description: Delete a specified home

Usage: /removehome <home_name>

homes

Description: Open the home management GUI

Usage: /homes

tpa

Description: Send a teleport request to another player

Usage: /tpa <username>

tpaccept

Description: Accept a teleport request from another player

Usage: /tpaccept <username>

tpdeny

Description: Deny a teleport request from another player

Usage: /tpdeny <username>

tpacancel

Description: Cancel an outgoing teleport request

Usage: /tpacancel

Messaging Commands

msg

Description: Send a private message to a player

Usage: /msg <username> <message>

reply

Description: Reply to the last received message

Alias: r

Usage: /reply <message>

ignoreplayer

Description: Ignore messages from a specific player

Usage: /ignoreplayer <player_name>

Alias:

  • /unignoreplayer <player_name> - Stop ignoring a player
  • /ignoredplayerlist - Show the list of ignored players

Economy Commands

pay

Description: Transfer money to another player

Usage: /pay <player_name> <amount>

balance

Description: Display your current balance

Usage: /balance

ah

Description: Access the AuctionHouse GUI to buy, sell, and search items

Usage:

  • /ah help
  • /ah sell
  • /ah search

quickshop

Description: Display QuickShop commands

Alias: cshop

Usage: /quickshop

kit

Description: Access the kit GUI

Usage: /kit

rewards

Description: Open the daily rewards GUI

Usage: /rewards

Utility Commands

autopickup

Description: Toggle automatic item pickup on or off

Usage: /autopickup

sb

Description: Toggle the scoreboard display on or off

Usage: /sb

veinminer

Description: Configure VeinMiner settings, which allow mining of ore veins

Alias: vm

Usage:

  • /veinminer mode
  • /veinminer pattern
  • /veinminer toggle

playerwarp

Description: Display PlayerWarp commands

Alias: pw

Usage: /playerwarp

discordsrv

Description: Provide instructions to link your Minecraft and Discord accounts

Usage:

  • /discordsrv help
  • /discordsrv link

trash

Description: Open a permanent item disposal GUI

Usage: /trash

Job and Quest Commands

jobs

Description: Open the job GUI

Usage: /jobs

quest

Description: Opens the quest GUI

Usage: /quest

Claim Commands

claim

Description: Open the claim management GUI to manage land claims and permissions

Usage: /claim

Alias:

  • /cl - Open claim list
  • /subdivideclaims - Create subdivisions within your claim
  • /extendclaim <amount> - Expand your claim by a specified number of blocks
  • /givepet <player_name> - Transfer ownership of your pet to another player

Interaction Commands

team

Description: Display a list of team-related commands

Usage: /team

togglepvp

Description: Toggle your PvP status on or off

Usage: /togglepvp

killrecaps

Description: Open the kill recap GUI

Usage: /killrecaps

Movement Commands

sit

Description: Sit at your current position

Usage: /sit

Alias:

  • /sit <toggle> - Sit on specific blocks

lay

Description: Lie down

Usage: /lay

crawl

Description: Crawl on the ground

Usage: /crawl

bellyflop

Description: Perform a belly flop

Usage: /bellyflop

spin

Description: Spin in place

Usage: /spin
